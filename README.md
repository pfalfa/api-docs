# API Docs Pfalfa
Documents API Pfalfa

## Usage

```bash
# Install depedencies
$ npm install

# Run application
$ npm run start
```

## Using Docker

```bash
# Build docker image
$ docker build -t pfalfa-api-docs .

# Run docker container
$ docker run --name pfalfa-api-docs -d -p 3030:3030 pfalfa-api-docs
```
